# Ring Try-On App

A React + TypeScript + Vite application for virtual ring try-on.

## Available Commands

### `npm run dev`
Starts the development server with hot module replacement (HMR). The app will be available at `http://localhost:5173` by default.

### `npm run build`
Builds the application for production. This command:
1. Runs TypeScript compiler to check types (`tsc -b`)
2. Creates an optimized production build in the `dist` folder

### `npm run lint`
Runs ESLint to check code quality and identify potential issues in the codebase.

### `npm run preview`
Locally preview the production build. Run this after `npm run build` to test the production bundle before deployment.
